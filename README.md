# HanaPath

HanaPath is a mobile-first Korean learning app built around four main tabs:

- Alphabet
- Vocabulary
- Sentences
- Listening

The app starts from Hangul basics and grows into level-based practice with review loops, quizzes, and the 5k vocabulary file.

## Run locally

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Main files

- `index.html`
- `app.js`
- `styles.css`
- `korean_5000_claude_ready.csv`

## Notes

- Progress is stored in the browser.
- The repo is meant to be static and easy to deploy to GitHub Pages or another static host.
